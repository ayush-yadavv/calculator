# Flutter Calculator

![Flutter Calculator](screen.png)

This is a simple calculator app built with Flutter that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (X)
- Division (/)
- Decimal Point (.)
- Clear (CLEAR)
- Equals (=)

## Getting Started
